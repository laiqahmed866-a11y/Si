# 🎂 Happy 22nd Birthday Slideshow

A beautiful birthday slideshow with gift box intro, music, confetti and floating hearts.

```

```

## 🚀 GitHub Pages Setup

1. Upload all files to a GitHub repository
2. Go to **Settings → Pages**
3. Select **Branch: main** → **/ (root)** → Save
4. Your link: `https://yourusername.github.io/repository-name`

## 📸 Adding Photos

1. Rename your 10 photos as: `photo1.jpg`, `photo2.jpg` ... `photo10.jpg`
2. Put them inside the `images/` folder
3. Push to GitHub — done!

## 🎵 Features

- 🎁 Gift box opening animation
